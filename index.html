Theme

Our main theme is comic book. We want the website to have a fun vibe and so, we will be using colors that are vibrant and bright.

We also want our page to have relevant images in the boxes and have the text show up when the user hovers (or touch for mobile) over any of the boxes.

We think that we can use animations that would show the common phrases thrown out in comics like "wham!" or "Kaboom."
